Tła i gradienty w CSS

Pojęcie Jednozadaniowy opis Przykład
Kolor tła Ustawia jednolity kolor tłą elementu. selektor { background-color: yellow; }
Obraz tła Dodaje obraz jako tło elementu. background-image: url("obraz.jpg");
Gradient liniowy Płynne przejście pomiędzy kolorami. background: linear-gradient(red, yellow);
Gradient kołowy Gradient rozchodzący się od środka. background: radial-gradient)blue, white);
Powtarzanie tłą Określa sposób powtarzania obrazu tła. background-repeat: no-repeat;
Pozycja tła Ustawia położenie obrazu tła. background-position: center;
